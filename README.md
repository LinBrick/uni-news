## 目录结构


```bash
├── cloudfunctions-aliyun      # 云函数
├── common                     # 常用的文件
│   │── api                    # 请求接口函数
├── components                 # 自定义组件
├── pages                      # 页面存放目录
├── static                     # 静态资源 (会被直接复制)
├── store                      # 全局 vuex store
├── unpackage                  # 编译后的文件存放目录
├── utils                      # 公用的工具类
├── App.vue                    # 入口页面
├── main.js                    # 入口文件 加载组件 初始化等
├── manifest.json              # 项目配置
├── pages.json                 # 页面配置
├── uni.scss                   # sass常用变量
```
